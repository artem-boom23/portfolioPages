React
React – библиотека JS которую я постоянно использую для разработки пользовательского интерфейса(UI) своих проектов. Все проекты пишу на языке TypeScript(т.к он является типизированным и поддерживает модули) и применяю архитектурный паттерн Atomic Design. Для стилизации своих работ я выбираю библиотеку компонентов MUI
Паттерны и алгоритмы
• Работа с алгоритмами и структурами данных: стек, очередь, связанные списки, деревья.
• Оптимизация сортировок, таких как: сортировка вставками, сортировка выбором, сортировка слиянием, пирамидальная сортировка, алгоритм быстрой сортировки.
